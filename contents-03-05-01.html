청구 관리

청구를 생성하고 관리합니다.

  • 청구관리하기

    청구관리에서는 출금 및 결제를 위한 기본 데이터들을 생성하고 관리할 수 있습니다.

  • 청구관리 화면

  • 검색 : 오른쪽 상단 검색버튼을 클릭하여 목록을 검색할 수 있으며, 검색기간을 기준으로 목록을 조회가능 합니다. 또한 분류기능을 통해 회원구분/상품/결제방식/결제수단/결제일을 기준으로 목록을 분류하여 조회할 수 있습니다.

    청구별 묶음보기 : 하나의 청구단위를 기준으로 청구 목록 조회하는 기능입니다. 하나의 청구단위 안에 여러가지 상품이 존재할 수 있습니다.

    상품별 펼쳐보기 : 청구단위가 아닌 상품 단위를 기준으로 청구 목록을 조회하는 기능입니다. 상품별 펼쳐보기는 상품 단위에서 금액을 일괄적으로 수정해야 하는 경우 유용하게 사용할 수 있습니다.

    엑셀 저장 : 엑셀 아이콘 클릭 시, 청구 목록에 대해 엑셀 파일로 다운로드할 수 있습니다.

    청구 생성 : “정기 청구” “추가 청구” 두가지 나뉩니다.

    정기 청구: 회원 등록 시, 청구 생성 방식을 자동으로 설정하지 않고 수동으로 설정한 경우에는 해당 버튼을 클릭하여 수동으로 청구를 생성하게 됩니다.

    추가 청구: 정기적으로 발생하는 수납/결제가 아닌 비정기적인 수납/결제 청구를 회원별로 생성할 수 있습니다. 회원을 검색하여 결제일(납부기간)을 선택하고 상품을 추가하여 추가 청구를 생성합니다.

    청구 수정: “청구 일괄” “대량 엑셀” 두가지로 나뉩니다.

    청구 일괄: 청구 결제일 등을 일괄적으로 수정하고 싶은 경우, 수정을 원하는 청구 건을 선택 후 일괄적으로 수정할 수 있습니다.

    대량 엑셀: 엑셀 파일을 이용하여 대량으로 청구를 수정할 수 있습니다.

    청구 삭제 : 삭제를 원하는 청구 건을 선택하여 삭제할 수 있습니다. 청구 삭제를 하더라도 발송된 청구서는 삭제되지 않습니다. 발송된 청구서 삭제를 원할 시에는 먼저 ‘청구서 발송 취소’ 진행 후 청구 삭제를 진행하여야 합니다.

    청구서 보기 : 청구 목록에서 원하는 청구를 선택 후 ‘청구서 보기’를 클릭하면해당 청구에 대한 청구금액과 상세 내역이 담긴 청구서를 확인할 수 있습니다. 해당 청구서 보기에서는 청구서를 인쇄 기능도 제공하고 있습니다.

    청구서: “보기” “발송 취소” “발송” 세가지로 나뉩니다.

    보기: 청구 목록에서 원하는 청구를 선택 후 ‘보기’를 클릭하면해당 청구에 대한 청구금액과 상세 내역이 담긴 청구서를 확인할 수 있습니다. 청구서 인쇄도 가능합니다.

    발송 취소 : 발송 취소를 원하는 청구 건을 선택 후 ‘청구서 발송 취소’를 클릭하게 되면, 해당 청구 건으로 발송된 청구서에 대해서 무효화가 처리되어 회원이 해당 청구서를 조회할 수 없게 됩니다.

    발송 : 청구 목록에서 원하는 청구서 발송을 원하는 청구를 선택 후 ‘발송’을 클릭하면 해당 청구 회원에서 청구서를 발송하게 됩니다.

  • 청구 생성-정기 청구

    청구관리에서 ‘청구생성-정기청구’ 클릭 시 청구를 생성하기 위한 창이 생성됩니다.

    검색 : 청구월을 기준으로 검색이 가능합니다. 검색 결과에는 해당 청구월의 청구 가능 회원 목록이 나타나고, 이미 청구 생성이 완료된 회원은 목록에 나타나지 않습니다.

    청구생성 : 검색을 통해 나타난 청구 가능 회원 목록에서 청구 생성을 원하는 회원을 선택 후 ‘청구생성’을 클릭하면 생성이 완료됩니다.

    TIP!

    청구 생성을 자동으로 설정한 회원에 한해

    • 매월 마지막 영업일, 다음달 청구 데이터가 일괄 생성되며 새롭게 등록된 회원은 매일 아침 청구 데이터가 자동으로 생성됩니다.

    TIP!

    청구 데이터 생성 및 수정은 아래 시간까지 가능합니다.

    • 자동결제(CMS)는 결제일 전 영업일 17시
    • 자동결제(실시간CMS, 카드, 휴대전화)는 결제일 당일 10시
    • 실시간CMS, 카드, 휴대전화의 경우 결제일을 당일로 설정한 경우, 즉시결제가 진행됩니다.

    대량 청구 수정

    청구수정 : 파일을 선택하여 업로드한 후 청구수정을 적용할 수 있습니다.

    수정 가능 청구 목록 다운로드 : 해당 월에 수정 가능한 청구 목록을 엑셀 파일 형식으로 다운받을 수 있습니다.

    청구 수정 및 관리하기

    청구관리 화면

    청구서보기 : ‘청구서 보기’를 클릭하면 해당 청구 건에 대한 청구금액 및 내역 등이 나타나는 청구서를 확인할 수 있습니다. 해당 청구서는 청구서를 발송했을 때 회원이 받게 되는 실제 청구서와 동일한 내용을 담고 있습니다.

    수정 : 상품 금액이나 수량 등 원하는 정보를 수정 후 해당 ‘수정’을 클릭하여 수정을 완료할 수 있습니다.

    청구 삭제 : ‘청구 삭제’를 클릭하여 해당 청구를 삭제할 수 있습니다.

    청구서 발송 : ‘청구서 발송’을 클릭하여 해당 청구에 대한 청구서를 회원에게 발송할 수 있습니다. 회원에게 발송되는 청구서에는 자동결제 청구서, 납부자결제 청구서 2가지가 있습니다. 자동결제 청구서는 자동 이체가 될 예정을 회원에게 알리기 위한 알림형 청구서입니다. 납부자 결제 청구서는 회원이 해당 청구서를 통해서 직접 결제를 진행 할 수 있도록 결제하기 버튼을 포함하는 청구서입니다.

    청구서 발송 이력 : 회원에게 발송한 청구서의 발송 완료 및 취소 이력을 해당 내역에서 확인할 수 있습니다.

    TIP!

    • 자동결제 청구서의 경우 SMS기반으로 발송되며 납부자 결제 청구서는 카카오 알림톡을 기반으로 발송되는 차이점이 있습니다.

    TIP!

    청구서 발송이 자동으로 설정된 경우 청구서 발송 기준 시간은 다음과 같습니다.

    • 자동결제(CMS, 카드, 휴대전화)의 경우 당일 10시경 익영업일 결제가 이뤄지는 청구 데이터의 청구서가 발송됩니다.
    • 가상계좌, 납부자결제의 경우 결제시작일 10시경 청구서가 발송됩니다.
