회원 등록하기

회원등록을 위한 입력정보 및 등록과정을 확인합니다.

  • 회원별 기본정보 및 계약정보(상품)를 등록합니다. 수납금액을 설정하고 고객으로부터 수취한 결제정보를 등록 한 후 청구생성 방식까지 설정하면 새로운 회원등록이 완료됩니다. 회원등록은 [회원 > 회원관리 > 회원등록]에서 진행할 수 있습니다.

  • 기본정보 입력

    회원명, 연락처 등의 회원의 기본정보를 입력합니다. (*표시가 있는 항목은 필수 입력 정보입니다.)

    회원유형 : 결제정보가 개인명의일 경우 ‘개인회원’을 법인명의일 경우 ‘사업자회원’을 선택합니다.

    회원번호 : 회원 관리를 위한 고유 번호를 입력합니다.

    • TIP!

    • ‘설정 > 서비스설정’ 메뉴에서 회원번호 자동/수동 설정이 가능합니다.

  • 회원구분 : 회원구분 값을 선택합니다. ‘미지정’을 선택하면 설정되지 않습니다.

    TIP!

    • ‘설정 > 서비스설정’ 메뉴에서 회원구분을 추가 또는 변경할 수 있습니다.

  • 휴대전화 : 회원의 연락처를 입력합니다.

  • 계약정보(상품) 설정

    회원에게 제공하는 상품정보를 선택하고 수납금액 설정합니다.

    상품추가 : 회원에게 제공하는 상품을 선택하고 상품금액, 수량 등 상세 정보를 입력하여 상품을 추가합니다.

    • TIP!

    • ‘설정 > 상품설정’ 메뉴에서 상품정보를 추가하거나 수정할 수 있습니다. 상품 등록 시 과세/면세 여부도 설정 가능합니다.

  • 결제정보 등록

    회원별 결제방식을 설정하고 결제정보를 등록합니다.

    결제방식 : ‘자동결제’를 선택하여 결제정보를 등록합니다.

    • TIP!

    • 납부자결제, 가상계좌, 직접결제에 대한 내용을 회원관리 상세 안내 파트에서 확인해보세요.

  • 약정일 : 회원과 약정한 결제일자를 선택합니다.

    재처리 : 재출금(승인) 방식을 자동/수동 선택할 수 있습니다.

    • TIP!

    • ‘설정 > 청구설정’ 메뉴에서 재처리 주기 및 횟수를 설정할 수 있습니다.

  • 청구기간 : 청구가능 기간을 설정합니다. 청구기간 내에서 청구생성이 가능합니다.

    결제주기 : 매월 청구한다면 ‘정기’를 특정월에만 청구한다면 ‘비정기’를 선택하고 해당월을 설정합니다.

    결제수단 : 회원의 결제수단을 선택합니다. ‘회원설정’을 선택하면 결제정보를 등록하지 않고 회원을 등록할 수 있습니다.

    • TIP!

    • ‘회원설정’을 통해 결제정보를 등록하지 않고, 회원에게 간편동의 메시지를 발송하여 결제정보를 직접 입력 받을 수 있습니다.

  • 간편동의 요청수단 : 간편동의 메시지 발송 정보를 입력합니다. ‘회원정보와 동일’을 선택하면 기본 정보에서 입력한 휴대전화 번호가 자동 입력됩니다.

    자동결제 청구서발송 : 청구서 발송 여부를 선택합니다.

    결제정보입력 : 회원의 결제정보를 입력하고 동의자료를 첨부합니다.

  • 청구정보 설정

    청구 데이터 생성과 청구서 발송 정보를 설정합니다.

    청구생성 : 청구생성을 ‘자동’으로 설정하면 청구데이터가 약정일에 맞춰 자동 생성되고, ‘수동’으로 설정하면 사용자가 직접 청구데이터를 생성해야 합니다.

    청구서발송 : 자동결제 청구서발송 대상 고객이라면 청구서 발송 방식을 ‘자동’ 또는 ‘수동’으로 설정할 수 있습니다. ‘자동’으로 설정하면 청구생성 시 청구서가 자동으로 발송됩니다.

    청구서 발송 수단 : 청구서는 이메일 또는 휴대전화로 발송 할 수 있습니다. ‘회원정보와 동일’을 선택하면 기본정보에 입력된 회원정보로 자동 입력됩니다.

  • 부가정보 설정

    현금수납 완료건에 대한 증빙유형을 설정하여 자동발행 처리합니다.

    대표증빙유형 : 현금수납 건에 대한 현금영수증, 전자세금계산서 정보를 설정하여 자동발행 처리합니다.

    • TIP!

    • 상세한 회원 등록과 관리방법을 회원관리 세부설명에서 확인해보세요.
