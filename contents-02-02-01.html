아이디 생성 및 접속하기

효성CMS+에서 사용자 아이디와 비밀번호를 설정합니다.

  • 효성CMS+를 최초 사용하기 위해서는 로그인 페이지에서 사용자 아이디 및 비밀번호를 생성해야 합니다.

    로그인 접속 화면

  • 아이디 만들기 : 최초로 효성CMS+에 접속하기 위해서는 사용자 아이디를 생성 하여야 합니다.

    아이디 : 이미 생성된 사용자 아이디를 입력합니다.

    비밀번호 : 이미 생성된 사용자 아이디의 비밀번호를 입력합니다.

    로그인 : 효성CMS+ 계정의 아이디와 비밀번호 입력 후 로그인을 클릭하여 프 로그램에 접속할 수 있습니다.

    아이디 저장 : 매번 아이디를 입력하는 번거로움 없이 아이디 저장을 통해 계정 아이디를 기억하여 로그인 절차를 줄일 수 있습니다.

    비밀번호 찾기 : 비밀번호를 기억하지 못할 때 비밀번호 찾기 절차를 진행할 수 있습니다.

  • 계약 아이디 확인

  • 최초에 효성CMS+의 사용자아이디를 만들기 위해서는 계약아이디를 인증하여야 합니다. 계약아이디 인증 후에 실제적인 효성CMS+ 사용을 위한 사용자아이디와 비밀번호를 설정하실 수 있습니다.

    계약아이디 : 효성CMS+ 서비스 계약 당시 발급된 아이디입니다.

  • 2차인증

  • 로그인 시에는 2차 인증 절차를 거치며, 계약 시 작성하신 담당자 휴대폰 번호 또는 이메일로 인증번호가 발송됩니다.

    • TIP!

    • 한 번 2차 인증을 진행하면 인증 결과가 브라우저에 저장되기 때문에 한 달 동안 동일한 브라우저에서는 2차 인증을 생략할 수 있습니다.
