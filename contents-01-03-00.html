결제신청

결제신청은 효성CMS+의 청구 생성을 통해 자동 또는 수동으로 가능합니다.

  • 청구 생성하기

    청구데이터를 자동으로 생성합니다.

    효성CMS+에서 회원등록 시 청구 정보의 ‘청구생성’을 ‘자동’으로 설정하면 약정일에 맞춰 청구데이터가 자동으로 생성됩니다. (기존 효성CMS 서비스에서 출금방식을 ‘자동’으로 이용했다면 업그레이드 된 효성CMS+에서도 데이터 이관 시 청구 생성이 ‘자동’으로 설정됩니다.)

  • 자동생성 기준

  • 일괄생성 : 매월 마지막 영업일 07:00 신청 완료 된 회원의 청구데이터가 약정일에 맞춰 자동 생성됩니다.(약정일이 휴일이면 다음 영업일로 생성)
  • 추가생성 : 매일(영업일) 07:00 신규 등록 된 회원이 있다면 해당 회원의 청구데이터가 약정일에 맞춰 자동 생성됩니다.(약정일이 휴일이면 다음 영업일로 생성)

TIP!

  • 당월 신규 등록 된 회원의 약정일이 이미 지난 경우는 다음달부터 자동 생성됩니다. 약정일이 지난 회원의 당월 청구를 생성하기 위해서는 ‘청구생성’ 을 통해 수동 생성해야 합니다.

  • 상세한 청구관리 화면과 기능을 확인 확인해보세요.

  • 수동 청구생성

    효성CMS+에서 회원등록 시 청구정보의 ‘청구생성’을 ‘수동’으로 설정한 회원은 사용자가 직접 청구데이터를 생성합니다.

  • 수동 청구 생성하기

    청구월 : 각 청구는 대상 청구월(귀속월)이 있습니다. 몇 월분 청구데이터를 생성할 것인지 선택하여 검색합니다.

    정기청구 : 해당 청구월에 청구건이 생성되지 않은 리스트가 보여집니다. 결제일 및 기본금액, 수량을 수정하고 생성하고자 하는 청구건을 선택하여 ‘청구생성’을 버튼을 누르면 청구가 생성됩니다.

    추가청구 : 정기청구 외 비정기적 청구는 추가청구를 생성하여 수납/결제합니다. (미수건은 추가청구가 아닌 미수관리 메뉴에서 처리해 주세요!)

TIP!

  • 실시간CMS, 카드, 휴대전화 결제 회원은 결제일을 당일로 청구생성하면 즉시결제가 이루어집니다. 납부자결제, 가상계좌 회원은 결제시작일을 당일로 청구생성하면 청구서가 즉시 발송됩니다.

  • 상세한 청구관리 화면과 기능을 확인 확인해보세요.

  • 청구정보 수정 및 삭제

    효성CMS+의 [청구 > 청구관리]에서 생성된 청구데이터를 조회하고 정보를 수정하거나 삭제할 수 있습니다.

    청구 수정 및 삭제

  • 청구수정

  • 대상 청구 선택 : 청구관리에 조회된 청구 중 수정이 필요한 청구 건을 더블클릭합니다.
  • 수정 : ‘수정’ 버튼을 눌러 필요한 정보를 수정하고 ‘저장’ 버튼을 누르면 완료됩니다.

  • 청구삭제

  • 대상 청구 선택 : 청구관리에 조회된 청구 중 삭제가 필요한 청구 건을 선택합니다.
  • 삭제 : 화면하단 ‘청구삭제’ 버튼을 눌러 청구를 삭제합니다.
