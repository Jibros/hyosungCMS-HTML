사용자아이디 만들기

최초 사용자아이디 생성 절차를 확인합니다.

  • 최초 접속을 위해서는 로그인 페이지에서 다음과 같은 절차로 사용자아이디를 생성합니다.

  • 아이디 만들기

    로그인 페이지에서 ‘아이디만들기’ 버튼을 클릭하고, 계약 시 수취한 계약아이디로 인증 후 사용자 아이디를 생성합니다.

    아이디만들기 : 로그인페이지에서 해당 버튼을 클릭하여 사용자 아이디를 생성합니다.

  • 계약아이디 확인

    로그인 페이지에서 ‘아이디만들기’ 버튼을 클릭하고, 계약 시 수취한 계약아이디로 인증 후 사용자 아이디를 생성합니다.

    서비스 계약 과정에서 카카오 알림톡으로 수취한 계약아이디와 비밀번호를 입력하여 계약자 인증을 진행합니다.

    계약아이디 : 계약아이디를 입력합니다.

    비밀번호 : 비밀번호를 입력합니다.

  • 사용자아이디 생성

    사용을 원하는 사용자아이디(마스터 아이디) 및 비밀번호, 기타 정보를 입력하고 아이디를 생성합니다.
