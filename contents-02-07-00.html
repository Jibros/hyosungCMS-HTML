미수 관리하기

미수 관리를 통해 결제를 자동 또는 수동으로 재처리합니다.

  • 자동 결제 재처리

    결제가 실패한 청구 건들을 자동으로 결재 재처리합니다.

    결제 실패한 청구 건을 자동으로 결제 재처리할 수 있습니다. 재처리 주기 및 횟수도 별도 설정 가능합니다.

  • 자동 결제 재처리 등록

    회원등록 시 결제정보의 ‘재처리’ 항목을 ‘자동’으로 설정하면 결제실패 시 자동으로 결제 재처리합니다.

  • 자동 재처리 기준 설정

    ‘설정 > 청구설정’ 메뉴의 미수관리 설정을 통해 자동 재처리 주기 및 횟수를 설정합니다.

    • 재결제 처리 자동주기 설정 : 결제 실패건을 자동 재결제 처리 할 주기를 설정합니다. 최소 2일 ~ 최대 15일 단위로 설정 가능합니다.
    • 재결제 처리 횟수 설정 : 결제 실패 시 최대 자동 재결제 처리 횟수를 설정합니다. 2회 또는 3회로 설정 가능합니다.

  • 결제 재처리 현황 확인

    ‘수납 > 수납관리’ 메뉴에서 재처리중인 청구건을 확인할 수 있습니다.

    • 재처리 현황 : 수납관리 리스트의 ‘미수처리상태’를 통해 재처리 진행중인 건을 확인할 수 있습니다. ‘재처리’로 표시됩니다.
    • 결제상태 : 재처리 결제현황은 ‘결제상태’를 통해 확인합니다.

  • 수동 결제 재처리

    결제가 실패한 청구 건들을 수동으로 결재 재처리합니다.

    [수납 > 미수관리]에서 결제를 실패한 청구 건을 확인하고 수동으로 결제 재처리를 할 수 있습니다.

    결제 재처리 신청(당월결제)

    결제가 실패한 미수내역을 조회하여 결제 재처리를 진행합니다.

    • 대상선택 : 청구월 기준 검색기간을 조회하면 해당 기간동안 발생한 결제실패(미수)를 조회하고 결제 재처리를 신청할 건을 선택합니다.
    • 결제일 : 재처리를 시도할 날짜를 선택합니다.
    • 결제 재처리 : ‘결제 재처리’ 버튼을 누르면 재처리 신청이 완료됩니다.

  • 결제 재처리 현황 확인

    • ‘수납 > 수납관리’ 메뉴에서 재처리 현황을 확인할 수 있습니다.
    • 재처리 현황 : 수납관리 리스트의 ‘미수처리상태’를 통해 재처리 진행중인 건을 확인할 수 있습니다. (‘재처리’로 표시)
    • 결제상태 : 재처리 결제현황은 ‘결제상태’를 통해 확인합니다.

  • 다음 청구 시 재청구

    결제 실패한 청구 건에 대해 다음 청구 시 재청구하도록 요청할 수 있습니다.

    [수납 > 미수관리]에서 결제 실패한 청구 건을 확인하고 다음 청구 시 재청구할 수 있습니다.

    • 결제가 실패한 미수내역을 조회하여 다음 청구 시 재청구합니다. 미납건 재청구 처리 시 해당월 청구데이터와 별도로 청구데이터가 생성됩니다.
    • 미납건 재청구일 : 미납이 발생한 청구월 다음달부터 선택 가능합니다. 선택한 일자로 미납 청구데이터가 생성되고 청구월은 해당 일자가 속해 있는 달로 설정됩니다.

  • 합산청구

    결제가 실패한 미수내역을 조회하여 다음 청구 시 재청구합니다. 합산청구 처리 시 해당월 청구데이터에 금액이 합산되어 청구데이터가 생성됩니다.

    • 합산청구월 : 미납이 발생한 청구월 다음 달부터 선택 가능합니다. 선택한 월에 해당 회원의 청구데이터 생성 시 미납금액이 합산됩니다.
