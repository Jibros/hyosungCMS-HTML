사용자 설정

사용자 계정을 추가하고 사용자별 권한을 부여합니다.

  • 사용자 설정에서는 마스터 아이디 외 서브 아이디를 발급할 수 있으며, 각 사용자 아이디의 권한을 설정할 수 있습니다.

    사용자 설정 화면

  • 사용자 추가 : 새로운 사용자 아이디를 추가합니다.

    • TIP!

    • [사용자설정] 메뉴는 마스터 아이디로 로그인 시에만 나타나는 메뉴입니다. 만약 화면에서 [사용자설정]이라는 메뉴가 보이지 않을 경우에는 서브 아이디가 아닌 마스터 아이디로 로그인 해보세요.

  • 사용자 등록

    사용자 등록을 위해 추가 사용할 계정의 아이디와 비밀번호, 사용자명, 사용자구분 등의 정보를 입력합니다.

    사용자ID : 등록할 사용자 아이디를 입력하고, 중복여부를 체크할 수 있습니다.

    비밀번호 : 등록할 사용자 아이디의 비밀번호를 입력합니다.

    사용자명 : 사용자명을 입력합니다.

    사용자구분 : 업체관리자 및 일반관리자를 선택할 수 있습니다.

    부서명 : 사용할 사용자의 부서명을 입력합니다.

    회원관리 : 회원관리 범위를 전체회원 또는 본인회원 중 선택할 수 있습니다.

    휴대전화 : 휴대전화 번호를 입력합니다.

    유선전화 : 유선전화 번호를 입력합니다.

    팩스번호 : 팩스 번호를 입력합니다.

    이메일 : 등록할 사용자의 이메일을 입력합니다.

  • 사용자별 권한 설정

    회원관리, 수납관리, 환경설정 등의 다양한 기능들에 대해서 사용자 아이디별 권한 범위를 설정할 수 있습니다.

    권한없음 : 해당 메뉴에 대해 접근권한을 부여받지 못합니다.

    조회 : 접근은 가능하나 등록, 수정, 삭제 등이 불가하고 단순한 조회만 가능합니다.

    관리 : 접근도 가능하고 등록, 수정, 삭제 등의 편집 행위가 모두 가능합니다.

    회원관리 : 회원 메뉴 기능 및 화면 접근에 대한 권한을 설정합니다.

    청구/수납관리 : 청구와 수납 메뉴 기능 및 화면 접근에 대한 권한을 설정합니다.

    환경설정 : 설정 메뉴 기능 및 화면 접근에 대한 권한을 설정합니다.

  • 사용자별 상세

    사용자 상세 화면에서 각 사용자의 기본정보 및 권한 설정을 확인하고 수정할 수 있습니다.

    삭제 : 해당 사용자 아이디를 삭제할 수 있습니다.

    수정 : 해당 사용자 아이디의 정보 및 권한을 변경하고 세부 정보를 수정할 수 있습니다.
