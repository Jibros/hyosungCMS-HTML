기본 설정하기

이용을 위해 기본으로 설정해야 할 사항을 확인합니다.

  • 효성CMS+ 서비스 이용을 위해 [설정]에서 다음과 같이 기본적으로 설정해야 할 사항들을 확인합니다.

  • 상품 설정하기 (설정 > 상품설정)

    회원에게 제공하는 상품을 등록합니다. 상품정보는 각 회원에게 수납할 금액을 구성하는 기본 단위입니다.

    상품 추가 : 새로운 상품을 추가합니다. 상품명, 금액을 입력하고 과세요율(과세/면세)을 선택하면 상품 등록이 완료됩니다.

    • TIP!

    • 매월 청구하는 금액이 변동되는 경우 0원 상품을 등록하고 청구데이터 생성 시 청구금액을 수정할 수 있습니다.

    • [설정 > 상품설정] 에서 자세한 상품설정 방법을 확인해보세요.

  • 서비스설정하기 (설정 > 서비스설정)

    서비스 이용에 필요한 정보를 설정합니다.

    회원구분설정 : 회원을 그룹핑하여 관리하기 위해 회원 구분값을 추가할 수 있습니다.

    수기수납 수단 : 효성CMS+가 제공하는 결제서비스가 아닌 직접 받은 수납을 수기수납처리하기 위한 수납명을 등록합니다.

    회원번호설정 : 회원등록 시 각 회원에게 설정되는 고유 회원번호 생성방식을 자동 또는 수동으로 설정합니다.

    • TIP!

    • 자체적으로 관리하는 회원코드가 있다면 회원번호설정을 ‘수동’으로 설정하여 통합하여 관리할 수 있습니다.

    • [설정 > 서비스설정] 에서 자세한 서비스 설정 내용을 확인해보세요.

  • 청구설정하기(설정 > 청구설정)

    청구데이터 생성과 연관된 기본값을 설정합니다.

    청구 기본 설정 : 효성CMS+를 통해 생성하는 청구데이터의 기준정보를 설정합니다.

    회원별 기본 설정 : 회원등록 시 설정하는 청구정보의 기본값을 설정합니다.

    미수관리 설정 : 미수 건 자동 재결제 처리 기준(주기/횟수)을 설정합니다.

    • TIP!

    • 매월 정해진 날짜에 고정 금액을 수납하는 경우 청구생성과 재결제처리를 ‘자동’으로 설정하면 보다 편리하게 서비스를 이용할 수 있습니다.

    • [설정 > 청구설정]에서 자세한 청구 설정 내용을 확인해보세요.
