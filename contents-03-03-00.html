메인 화면 설명

청구, 수납 상태에 대한 간략한 상태를 메인에서 확인합니다.

  • 메인 화면 설명

    전체회원 : 현재 등록된 전체회원 수를 표시합니다. 해당 숫자에는 삭제회원을 포함하지 않습니다.

    청구 가능 회원 : 현재 청구가능한 회원의 수를 표시합니다.

    청구 대기 : 아직 청구 처리되지 않고 대기 상태에 있는 건을 표시합니다.

    청구 완료 : 청구 처리가 완료된 건을 표시합니다.

    (수납)대기 : 아직 결제 처리되지 않고 대기 상태에 있는 건을 표시합니다.

    (수납)완납 : 결제 처리되어 결제 성공한 건을 표시합니다.

    (수납)미납 : 결제 처리했지만 결제 실패한 건을 표시합니다.

    수납일정 : 생성된 청구데이터를 캘린더에 표시하고 해당 일의 청구/수납 건수, 청구/수납 금액에 대해 빠르게 파악할 수 있게 요약 내역을 나타내줍니다.

    전자금융알리미 : 사용자가 효성CMS+를 통해 처리한 업무를 순차적으로 표시해줍니다.
