서비스 기본 설정하기

회원, 수납 등과 관련된 기본 설정을 등록합니다.

  • 서비스 설정에서는 회원구분, 회원번호설정, 수기수납수단 등 서비스 기본 설정을 등록할 수 있습니다.

    회원구분

    회원구분은 등록된 여러 회원을 그룹화하여 그룹별로 보다 쉽고 통합적으로 관 리할 수 있도록 하는 설정입니다.

    추가 : 회원구분을 추가하기 위해서는 해당 버튼을 클릭합니다.

    이동 : 그룹별 회원을 통합적으로 이동시킬 수 있습니다.

    • TIP!

    • 회원구분설정을 통해 회원을 그룹화하게 되면 그룹별 정보 수정 및 출금 신청 등의 업무를 보다 통합적으로 빠르게 처리할 수 있습니다.

  • 수기수납수단

    수기수납은 효성CMS+가 제공하는 결제서비스가 아닌 타 결제 수단으로 수납 받았을 경우 해당 건을 수기로 등록하여 처리하는 것을 의미합니다.

    추가 : 해당 결제 수단명을 입력하여 수기수납 수단을 추가할 수 있습니다.

  • 기본설정

    회원등록번호 설정 : 회원등록 시, 각 회원에게는 고유의 회원번호가 생성되는데 해당 회원번호의 생성방식을 자동 또는 수동으로 설정할 수 있습니다.

    자동 : 자동으로 설정할 경우, 회원번호가 임의로 자동 부여됩니다.

    수동 : 수동으로 설정할 경우, 회원 등록 시 회원번호를 수동으로 직접 입력할 수 있습니다.

    • TIP!

    • 효성CMS+ 외 자체적으로 관리하는 고객코드가 있다면 회원등록번호를 ’수동’으로 설정하여 통합 운영할 수 있습니다.

  • 직인 : 직인을 등록해 놓으면 직인이 필요한 영수증, 거래내역서 등 회원에게 발급하는 각종 서류에 직인이 자동 날인되어 업무를 효율적으로 처리할 수 있습니다. 직인 등록은 jpeg, png, gif 등의 이미지 파일만 가능합니다.

  • 가상계좌 예금주명 설정

    가상계좌 예금주명 : 각 회원마다 발급되는 입금전용 가상계좌의 예금주명을 말하며 설정을 통해 표현방식을 변경할 수 있습니다.

    회원명 : 표시/미표시 설정으로 예금주명 내 회원명의 노출여부를 변경할 수 있습니다.

    업체명 표시위치 : 회원명 앞/뒤 설정으로 업체명의 표시위치를 변경할 수 있으며 업체명은 회원명과 달리 예금주명 내 필수적으로 들어가야 합니다.
